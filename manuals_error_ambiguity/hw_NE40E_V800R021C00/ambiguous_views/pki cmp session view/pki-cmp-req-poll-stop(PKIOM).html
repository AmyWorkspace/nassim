cmp poll-request stop

Function

The cmp poll-request stop command cancels the CMP requests that are being polled.

Format

cmp poll-request stop

Parameters

None

Views

PKI CMP session view, VS PKI CMP session view

Default Level

2: Configuration level

Task Name and Operations

Task Name Operations
pki write

Usage Guidelines

Usage Scenario

If a client sends a certificate-related request to CA server but the CA server does not respond immediately, the client polls the request at a specific interval until the server responds. If the CA server does not respond for a long period, you can cancel the request that is polled

Example

# Cancel the CMP request that is polled for CMP session session1.
<HUAWEI> system-view
[~HUAWEI] pki domain domain1
[*HUAWEI-pki-domain-domain1] pki cmp session session1
[*HUAWEI-pki-domain-domain1-pki-cmp-session-session1] cmp poll-request stop
