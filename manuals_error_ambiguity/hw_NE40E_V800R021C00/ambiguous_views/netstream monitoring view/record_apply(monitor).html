apply record

Function

The apply record command applies the NetStream flexible flow service in the monitoring view.

The undo apply record command cancels the NetStream flexible flow service in the monitoring view.

By default, the NetStream flexible flow service is not applied in the monitoring view.

Format

apply record record-name

undo apply record [ record-name ]

Parameters

Parameter Description Value
record-name

Specifies the name of a NetStream flexible flow.

The value is a string of 1 to 32 case-sensitive characters, spaces not supported.

Views

NetStream IPv6 monitoring view, NetStream monitoring view

Default Level

3: Management level

Task Name and Operations

Task Name Operations
netstream write

Usage Guidelines

Usage Scenario

After you configure a flexible flow, run the apply record command to apply the flexible flow service to the monitoring view. Then, this service is applied to an interface through the monitoring view. Packets on this interface are sampled and output as the flexible flow.

Prerequisites

Before applying a flexible flow, you must run the match or collect command to configure matching rules or collected fields. Otherwise, the flexible flow cannot be applied.

Precautions

Only one flexible flow cannot be applied in the IPv4 or IPv6 monitoring view.

Flexible flow packets can be output only in the V9 format.

  • If the ip netstream export version command is run in the system view to specify the V5 or IPFIX packet output format and the V5 or IPFIX format is also specified in the NetStream monitoring service view, the apply record command does not take effect.
  • If the ipv6 netstream export version command is run in the system view to specify the IPFIX format and the IPFIX format is also specified in the NetStream monitoring service view, the apply record command does not take effect.

Example

# Apply a flexible flow in the NetStream monitoring view.
<HUAWEI> system-view
[~HUAWEI] ip netstream record tt
[*HUAWEI-record-tt] collect input packets 64
[*HUAWEI-record-tt] commit
[~HUAWEI-record-tt] quit
[~HUAWEI] ip netstream monitor aaa
[*HUAWEI-monitor-aaa] apply record tt
