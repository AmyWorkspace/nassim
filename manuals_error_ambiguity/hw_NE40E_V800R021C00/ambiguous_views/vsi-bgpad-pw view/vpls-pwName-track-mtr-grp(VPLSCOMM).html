track monitor-group (VSI-LDP-PW view)

Function

The track monitor-group command configures service VSI PWs to track an L2VPN monitoring group.

The undo track monitor-group command disables service VSI PWs from tracking an L2VPN monitoring group.

By default, no service VSI PW tracks an L2VPN monitoring group.

Format

track monitor-group group-name

undo track monitor-group group-name

Parameters

Parameter Description Value
group-name

Specifies the name of an L2VPN monitoring group.

The value is a string of 1 to 31 case-sensitive characters without spaces.

Views

vsi-bgpad-pw view, VSI-LDP-PW view

Default Level

2: Configuration level

Task Name and Operations

Task Name Operations
l2vpn write

Usage Guidelines

Usage Scenario

To configure service VSI PWs to track an L2VPN monitoring group, run the track monitor-group command. After the configuration is complete, if the service VSI PWs tracking the L2VPN monitoring group go Up before the protocol VSI PWs bound to the monitoring group. After the protocol VSI PWs go Up, the service VSI PWs are unblocked after a specified hold-off timer expires.

Prerequisites

A PW to a peer has been specified in a VSI.

Precautions

The configuration of service VSI PWs to track an L2VPN monitoring group must be performed on both the master and backup PEs.

Example

# Configure a service VSI PW to track an L2VPN monitoring group.
<HUAWEI> system-view
[~HUAWEI] mpls
[*HUAWEI-mpls] quit
[*HUAWEI] mpls l2vpn
[*HUAWEI-l2vpn] quit
[*HUAWEI] vsi aa
[*HUAWEI-vsi-aa] bgp-ad
[*HUAWEI-vsi-aa-bgpad] vpls-id 100:1
[*HUAWEI-vsi-aa-bgpad] peer 1.1.1.1 pw pw1
[*HUAWEI-vsi-aa-bgpad-pw1] track monitor-group group1
