mldp p2mp (MPLS traffic statistics view)

Function

The mldp p2mp command enables mLDP P2MP traffic statistics collection.

The undo mldp p2mp command disables mLDP P2MP traffic statistics collection.

By default, this function is disabled.

Format

mldp p2mp

undo mldp p2mp

Parameters

None

Views

MPLS traffic statistics collection view, MPLS traffic statistics view

Default Level

2: Configuration level

Task Name and Operations

Task Name Operations
mpls-ldp write

Usage Guidelines

Usage Scenario

mLDP P2MP statistics collection helps you query and monitor traffic transmitted on mLDP P2MP tunnels, which improves maintainability. To enable mLDP P2MP traffic statistics collection, run the mldp p2mp command.

Example

# Enable mLDP P2MP traffic statistics collection.
<HUAWEI> system-view
[~HUAWEI] mpls
[*HUAWEI-mpls] quit
[*HUAWEI] mpls ldp
[*HUAWEI-mpls-ldp] quit
[*HUAWEI] mpls traffic-statistics
[*HUAWEI-traffic-statistics] mldp p2mp
