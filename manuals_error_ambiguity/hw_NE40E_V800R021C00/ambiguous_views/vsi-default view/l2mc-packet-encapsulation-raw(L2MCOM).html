l2-multicast protocol-packet encapsulation raw

Function

The l2-multicast protocol-packet encapsulation raw command is used to reserve the original tag and add a new tag to the IGMP Report packets when the packets are forwarded from L2VE sub-interface to L3VE QinQ termination sub-interface.

The undo l2-multicast protocol-packet encapsulation raw command restores the default configuration.

By default, the IGMP snooping does not reserve the tag in the IGMP Report packets.

Format

l2-multicast protocol-packet encapsulation raw

undo l2-multicast protocol-packet encapsulation raw

Parameters

None

Views

VSI-AUTO view, VSI-BD view, VSI-DEFAULT view, VSI-STATIC view

Default Level

2: Configuration level

Task Name and Operations

Task Name Operations
l2mc write

Usage Guidelines

Usage Scenario

IGMP Snooping function supports receiving IGMP packets with one tag from VPLS PW or VLL PW. When the packets are forwarded from L2VE sub-interface to L3VE QinQ termination sub-interface, the one tag packets cannot be processed. To solve the problem, run the l2-multicast protocol-packet encapsulation raw command to reserve the original tag and add a new tag to the IGMP packets, so that the packets can be forwarded from L2VE sub-interface to L3VE QinQ termination sub-interface.

Example

# Enable the function to reserve the original tag and add a new tag to the IGMP Report packets on VSI view.
<HUAWEI> system-view
[~HUAWEI] mpls
[*HUAWEI-mpls] quit
[*HUAWEI] mpls l2vpn
[*HUAWEI-l2vpn] quit
[*HUAWEI] igmp snooping enable
[*HUAWEI] vsi vsi1
[*HUAWEI-vsi-vsi1] pwsignal ldp
[*HUAWEI-vsi-vsi1-ldp] vsi-id 1
[*HUAWEI-vsi-vsi1-ldp] quit
[*HUAWEI-vsi-vsi1] l2-multicast protocol-packet encapsulation raw
