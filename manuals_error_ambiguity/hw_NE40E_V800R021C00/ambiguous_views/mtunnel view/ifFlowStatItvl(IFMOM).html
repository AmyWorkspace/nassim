set flow-stat interval (interface view)

Function

The set flow-stat interval command sets the interval at which traffic statistics are collected on an interface.

The undo set flow-stat interval command restores the default interval.

By default, the interval for collecting traffic statistics on an interface is 300 seconds.

Format

set flow-stat interval interval

undo set flow-stat interval

Parameters

Parameter Description Value
interval

Specifies the interval at which traffic statistics are collected on an interface.

The value is an integer ranging from 10 to 600 and must be a multiple of 10, in seconds.

Usage Guidelines

Usage Scenario

In port extension scenarios, to set the interval at which traffic statistics are collected on an interface, run the set flow-stat interval command in the port extension interface view on a master. The configuration is then delivered to the AP and takes effect on the AP's external communication interface mapped to the port extension interface. Similarly, run the set flow-stat interval command in the port extension trunk interface view to set an interval for collecting traffic statistics on an interface. The configuration is delivered to the AP and takes effect on the AP's external communication interfaces mapped to the port extension trunk member interfaces.

Prerequisites

Before running this command, run the role master command in the extended view of the port extension interface to set the current device as the master device.

Precautions

  • The new interval takes effect after the original interval expires. Traffic statistics are updated in the second cycle after the new interval takes effect.
  • The interval at which interface traffic statistics are collected cannot be configured on the management interface.

The interval configured in the port extension interface view takes precedence over that configured in the AP view.

Example

# Set the interval at which traffic statistics are collected on GE 1/0/1 to 120 seconds.
<HUAWEI> system-view
[~HUAWEI] interface GigabitEthernet 1/0/1
[~HUAWEI-GigabitEthernet1/0/1] set flow-stat interval 120
