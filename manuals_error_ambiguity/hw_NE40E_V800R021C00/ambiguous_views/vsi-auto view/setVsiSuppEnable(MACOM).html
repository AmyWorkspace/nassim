suppression enable(VSI view)

Function

The suppression enable command enables traffic suppression for a VSI.

The undo suppression enable command disables traffic suppression for a VSI or VLAN.

By default, traffic suppression for a VSI is disabled.

Format

suppression { inbound | outbound } enable

undo suppression { inbound | outbound } enable

Parameters

Parameter Description Value
inbound

Indicates that the upstream traffic is suppressed.

-
outbound

Indicates that the downstream traffic is suppressed.

-

Views

VSI-AUTO view, VSI-DEFAULT view, VSI-IVSI-BEVPNINTERWORK view, VSI-STATIC view, VSI view

Default Level

2: Configuration level

Task Name and Operations

Task Name Operations
mac write

Usage Guidelines

Usage Scenario

The suppression enable command enables traffic suppression for a VSI.

If traffic suppression has been enabled on an interface, traffic suppression is automatically enabled for the VSI bound to the interface, with no need to run this command in the VSI view.

Example

# Enable upstream traffic suppression for VSI huawei.
<HUAWEI> system-view
[~HUAWEI] mpls
[*HUAWEI-mpls] quit
[*HUAWEI] mpls l2vpn
[*HUAWEI-l2vpn] quit
[~HUAWEI] vsi huawei
[*HUAWEI-vsi-huawei] suppression inbound enable
