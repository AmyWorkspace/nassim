igmp-snooping transparent-vlan

Function

The igmp-snooping transparent-vlan command configures interfaces bound to a VSI to transparently transmit IGMP messages with VLAN tag.

The undo igmp-snooping transparent-vlan command cancels the configuration.

By default, the interfaces bound to a VSI cannot transparently transmit received IGMP messages with VLAN tag.

Format

igmp-snooping transparent-vlan

undo igmp-snooping transparent-vlan

Parameters

None

Views

VSI-AUTO view, VSI-DEFAULT view, VSI-STATIC view

Default Level

2: Configuration level

Task Name and Operations

Task Name Operations
l2mc write

Usage Guidelines

Usage Scenario

When IGMP messages with VLAN tag need to be transparently transmitted by the main interfaces bound to a VSI, run the igmp-snooping transparent-vlan command.

Prerequisites

The igmp-snooping enable command is configured in the VSI.

Precautions

The igmp-snooping transparent-vlan command cannot be run if one of the following operations is configured:

  • The igmp-snooping proxy command is configured in the VSI.
  • The igmp-snooping querier enable command is configured in the VSI.
  • The igmp-snooping report-suppress command is configured in the VSI.
  • The VSI is bound to an AC sub-interface.

Example

# Configure the main interfaces bound to the VSI 1 to transparently transmit IGMP messages with VLAN tag.
<HUAWEI> system-view
[~HUAWEI] mpls
[*HUAWEI-mpls] quit
[*HUAWEI] mpls l2vpn
[*HUAWEI-l2vpn] quit
[*HUAWEI] vsi 1
[*HUAWEI-vsi-1] pwsignal ldp
[*HUAWEI-vsi-1-ldp] vsi-id 200
[*HUAWEI-vsi-1-ldp] quit
[~HUAWEI-vsi-1] igmp-snooping transparent-vlan
